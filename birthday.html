FAQ

Add widgets ▶
Widgets can be added to your Home screen by using the Widget Gallery of your iPhone/iPad. You can find detailed instructions here:

https://support.apple.com/en-us/118610

Widget not in gallery ▶
It happens to some users that the birthday widget is not displayed in the iOS/iPadOS at first.

What can I do?

1. First of all, please make sure that you have created at least one birthday in the app.

2. Try to reload your event list by swiping down.

3. If the widget still does not appear in the gallery, please try to restart your device.

4. Lastly, you can try reinstalling the app if all other points fail.

Empty widget ▶
If a widget doesn't show any content at all (solid color), please try to restart the app and add the widget to your Home Screen again. If this does not solve the problem, please try to restart your device.
Purchase Error ▶
If an error occurs during the transaction, it is usually because the connection to the App Store has been interrupted. Therefore, please make sure that you are connected to the Internet and have access to the App Store. Also make sure that you have the necessary permissions to make purchases from the App Store. You can also visit the App Store Support page for more information.
Pro version on multiple devices ▶
Can I use the Pro version on all my devices after purchase?

Of course! Just open the apps purchase screen and tap HELP -> Restore. You should make sure that you are signed in with the same Apple ID on all your devices.

Incorrect date of birth (Bug) ▶
In a few cases it happened that the imported date of birth differs from the correct date of the contact (+-1 day). In this case please update the app (>1.4.8) and import the affected contacts again (overwritten).

A few users were affected by this bug, which has been finally fixed in app version 1.4.9.

Sorry about the circumstances.
